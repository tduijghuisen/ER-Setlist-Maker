Inloggen

Lichte beveiliging voor deze beheerpagina. De echte schrijfbeveiliging zit in je GitHub-token.

GitHub-token

Eenmalig instellen. De token blijft alleen in deze browser (localStorage) en wordt nooit op de site geplaatst.

Hoe maak ik een token aan?
  1. Ga naar GitHub → Fine-grained tokens (ingelogd als eigenaar van de repo).
  2. Bij Repository access: kies Only select repositories → ER-Setlist-Maker.
  3. Bij Permissions → Repository permissions: zet Contents op Read and write.
  4. Maak de token aan, kopieer hem (github_pat_…) en plak hierboven.

Agenda · optredens

Voorbije optredens (datum in het verleden) verdwijnen automatisch uit “Upcoming shows” en verschijnen onder “Bekijk historie”. De site sorteert alles op datum.

Na publiceren werkt de site over ±1 minuut bij (GitHub Pages bouwt opnieuw).

Video's · YouTube

Plak een YouTube-link of het video-ID. De volgorde hier bepaalt de volgorde op de site.

Na publiceren werkt de site over ±1 minuut bij.

Repertoire

Chord sheets, lyrics en songgegevens. Klik op een titel om het nummer uit/in te klappen. Exporteer per nummer naar PDF. Laden, bewerken en publiceren vereist een geldige GitHub-token.

Setlists

Stel per optreden een setlist samen uit het repertoire en exporteer naar PDF — als overzicht of als volledig boekje met alle chord sheets.

Regels met “SET 1/2/3” (of Encore/Pauze) worden koppen. Titels worden automatisch aan het repertoire gekoppeld; niet-gevonden titels kun je daarna handmatig koppelen.
